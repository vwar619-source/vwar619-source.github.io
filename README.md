# Vignesh Tax Website

A professional tax services website built with HTML, CSS, and JavaScript.

## Project Structure

```
vignesh-tax-website/
├── index.html              # Homepage
├── about.html              # About page
├── contact.html            # Contact page with form
├── posts/
│   └── post-1.html        # Blog post example
├── css/
│   └── style.css          # Styling
├── js/
│   └── script.js          # JavaScript functionality
└── assets/
    └── images/            # Add your photos & logos here
```

## Features

- **Responsive Design**: Mobile-friendly layout that works on all devices
- **Navigation**: Easy-to-use navigation menu across all pages
- **Contact Form**: Fully functional contact form with validation
- **Blog Section**: Ready for blog posts
- **Professional Styling**: Clean and modern design

## Pages

1. **Home (index.html)** - Landing page with welcome message
2. **About (about.html)** - Information about services and mission
3. **Contact (contact.html)** - Contact form for client inquiries
4. **Blog (posts/post-1.html)** - Sample blog post

## Getting Started

1. Clone or download this repository
2. Open `index.html` in your web browser
3. Replace placeholder content with your information
4. Add your logo and photos to the `assets/images/` folder
5. Deploy to your hosting service

## Customization

- Edit the HTML files to change page content
- Modify `css/style.css` to customize colors and styling
- Update `js/script.js` for additional functionality
- Replace the placeholder images with your own
